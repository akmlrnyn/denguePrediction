# Dengue Risk Early Warning System

Program simulasi sistem peringatan dini risiko penyebaran Demam Berdarah Dengue (DBD) berbasis terminal menggunakan bahasa C.


## SDG

SDG 3 — Good Health & Well-being


## Features

- House Index (HI)
- Container Index (CI)
- Breteau Index (BI)
- Dengue Risk Index (DRI)
- Risk Classification
- Intervention Simulation
